* 华为军事化风格，主要体现在目标自上而下，每层会与上层充分沟通，但不会过度挑战上层指示。这种风格造就了员工狼性、高效行动、忠诚执行。强大执行力是国内外所有CEO期望的，毕竟绝大多数人没有CEO的高瞻远瞩，很多时候更需要团队忠诚的执行。
